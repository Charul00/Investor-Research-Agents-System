# Architecture Diagrams

Editable Excalidraw board:

- `investment-research-architecture.excalidraw`

Open the file at `https://excalidraw.com` using **Open > Select file from device**.

The board contains five frames that map directly to the assignment requirements:

- `1. System Architecture`
- `2. Research Data Flow`
- `3. Database ER Diagram`
- `4. AI Orchestration Flow`
- `5. Multi-Tenant Isolation Flow`

The generator script is included so the board can be regenerated:

```bash
node docs/diagrams/generate-architecture-excalidraw.mjs
```
